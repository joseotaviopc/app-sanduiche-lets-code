# Passo Inicial

1. Clone esse repositório.
```
git clone https://github.com/joseotaviopc/app-sanduiche-lets-code.git
```

2. Acesse a pasta do projeto.

3. Instale as dependências.

```
npm install
```

4. Execute a aplicação em modo de desenvolvimento.

```
npm run dev
```
